# Шаблон ДКП (договор купли-продажи ТС)

`dkp-template.docx` — шаблон договора с метками для автозаполнения (синтаксис
[docxtemplater](https://docxtemplater.com/): `{поле}` — значение, `{#флаг}…{/флаг}` —
блок «если истина», `{^флаг}…{/флаг}` — блок «если ложь»).

Поддерживаются обе стороны и как **физлицо**, и как **юрлицо/ИП**.

`dkp-sample.pdf` — пример заполнения (Продавец — ООО, Покупатель — физлицо).
`_generate_template.py` — скрипт-генератор шаблона (python-docx).

## Поля

### Шапка
| Поле | Значение |
|------|----------|
| `city` | город составления |
| `day`, `month`, `year` | дата (например `19`, `июня`, `2026`) |

### Стороны: `seller_*` (Продавец) и `buyer_*` (Покупатель)
Флаг `{seller_org}` / `{buyer_org}` = `true`, если сторона — организация/ИП.

**Если организация/ИП** (`*_org = true`):
| Поле | Значение |
|------|----------|
| `*_org_name` | наименование (ООО «…», ИП Фамилия И.О.) |
| `*_ogrn` | ОГРН / ОГРНИП |
| `*_inn` | ИНН |
| `*_kpp` | КПП (у ИП — пусто/прочерк) |
| `*_signer_role` | кто подписывает (директора, ИП) |
| `*_signer_fio` | ФИО подписанта |
| `*_basis` | основание (Устава, свидетельства) |

**Если физлицо** (`*_org = false`):
| Поле | Значение |
|------|----------|
| `*_fio` | ФИО полностью |
| `*_birth` | дата рождения |
| `*_birthplace` | место рождения |
| `*_pasp_series`, `*_pasp_number` | паспорт: серия, номер |
| `*_pasp_issued_by`, `*_pasp_issued_date` | кем и когда выдан |
| `*_pasp_code` | код подразделения |

**Общие для обеих форм:**
| Поле | Значение |
|------|----------|
| `*_address` | адрес (юр. или регистрации) |
| `*_end` | окончание рода: `ое`/`ый`/`ая` (склейка «именуем{}…», «зарегистрированн{}…») |
| `*_short` | короткая подпись: `Иванов И.И.` |

### Автомобиль (из СТС/ПТС)
`car_brand`, `car_vin`, `car_type`, `car_category`, `car_year`, `car_engine`,
`car_chassis`, `car_body`, `car_color`, `car_power`, `car_volume`, `car_plate`,
`pts_series`, `pts_number`, `pts_issued`, `sts_series`, `sts_number`

### Цена
`price_num` (цифрами), `price_words` (прописью)

## Что заполняется откуда
- **Паспорт продавца/покупателя** → блок `seller_*` / `buyer_*` (физлицо).
- **СТС + ПТС автомобиля** → блок `car_*`, `sts_*`, `pts_*`.
- Цена, город, дата, форма стороны (физлицо/юрлицо) — вводятся в мини-аппе.
